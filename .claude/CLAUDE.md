@./CLAUDE.local.md

## 言語設定

**重要**: Claude Codeとのすべてのやり取りは日本語で行ってください。コード内のコメント、commit メッセージ、説明、エラーメッセージなど、すべての出力を日本語で提供してください。

## 作業スタイル

- コード変更前には必ず既存コードを読んで理解すること
- 変更は最小限にとどめ、必要な部分のみ修正
- 大きな変更の前には計画を提示して承認を得ること

## Git コミットメッセージ

フォーマット: `<type>: <subject>`

Types:

- feat: 新機能
- fix: バグ修正
- refactor: リファクタリング
- docs: ドキュメント
- test: テスト
- chore: その他

例: `feat: ユーザー認証機能を追加`

## コーディング規約

### Python

- PEP 8に準拠
- 型ヒントを使用
- docstringは日本語で記述

### JavaScript/TypeScript

- ESLint/Prettierの設定に従う
- コメントは日本語で記述

## エラーハンドリング

- エラーが発生したら、必ずユーザーに報告
- 推測で進めず、不明点は質問すること
- ログ出力は日本語で記述

## セキュリティ

- 機密情報(API キー、パスワード等)はコミットしない
- .envファイルは常に.gitignoreに含める
- セキュリティに関わる変更は必ず確認を取る
